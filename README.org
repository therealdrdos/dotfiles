This is the place where I store my personal config files.

* Directory layout
#+begin_example
dotfiles/
├── alacritty/
│   ├── alacritty.toml       Main Alacritty configuration
│   ├── local.toml.template  Template for local overrides
│   ├── themes/              Solarized theme variants
│   └── README.org           Alacritty-specific documentation
├── emacs/
│   ├── init.el              Main Emacs configuration
│   ├── lisp/                Local Emacs packages
│   └── README.org           Emacs-specific documentation
├── lsd/
│   ├── config.yaml          lsDeluxe configuration
│   ├── colors.yaml          Custom Solarized theme
│   └── README.org           lsd-specific documentation
├── portage/
│   ├── make.conf            Gentoo package manager configuration
│   ├── package.use/         USE flags for specific packages
│   ├── package.accept_keywords/  Testing packages
│   └── README.org           Portage-specific documentation
├── tmux/
│   ├── .tmux.conf           Tmux configuration
│   └── README.org           Tmux-specific documentation
└── README.org               This file
#+end_example

* Rights to use
Feel free to include the dotfiles in your projects or to get inspiration from them. However, I won't provide any warranty on the quality of them.
They are my personal files and I try to keep them up to date with general security/implementation recommendations, but there is no guarantee on that.

* Quick start
#+begin_src bash
  git clone https://github.com/therealdrdos/dotfiles ~/Somewhereonyourmachine
#+end_src  
  Consult the readme files for each conf you want to use, there may be specific instructions.
  Or use some app like GNU stow to manage them.
